★계산기 프로그램 구현 ()
-소괄호를 파악하여 그 부분을 먼저 연산한다.
-연산자의 우선순위를 근거로 연산의 순위를 결정한다.
-수식을 이루는 피연산자는 한자리 숫자로만 이뤄진다고 가정(이렇게 제한을 둔 이유는 학습의 포커스를 모으기 위함)

수식 표기법
-중위 표기법		5+2/7
-전위 표기법		+5/2 7
-후위 표기법		5 2 7/+

중위 표기법은 연산순서에 대한 정보가 담겨있지 않음.

따라서 지금 구현할 계산기의 연산과정은
1. 중위 표기법의 수식을 후위 표기법의 수식으로 바꿈
2. 후위 표기법으로 바뀐 수식을 계산하여 그 결과를 얻는다.


★새롭게 배운 함수
- void memset(void * ptr, int value, size_t num);   ->string.h
	ptr -> 채우고자하는 메모리의 시작 포인터
	value -> 메모리에 채우고자 하는 값
	num -> 채우고자하는 바이트의 수
쓰는 이유
연속된 메모리에 특정한 값을 넣을 때 사용 for문보다 더 빠른 속도가 나올 수 있음.

- int isdigit(int c);   ->ctype.h
	c -> 검사할 문자 또는 아스키 값
	반환값 -> 문자가 0~9 사이에 속하면 TRUE, 아니면 FALSE 반환
