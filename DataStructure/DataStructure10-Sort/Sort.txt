정렬 알고리즘의 성능 평가 근거
1. 비교의 횟수 - 두 데이터간의 비교연산 횟수
2. 이동의 횟수 - 위치의 변경을 위한 데이터의 이동횟수
빅-오 => 비교의 횟수

1.버블 정렬
제일 큰 값을 맨 뒤로 보내는 방식
비교의 횟수 => O(n^2) 
이동의 횟수 => O(n^2) 

2.선택 정렬
정렬 순서상 가장 앞서는 것을 선택해서 가장 왼쪽으로 이동시키고, 원래 그 자리에 있던 데이터는 
빈자리에 가져다 놓는다.
비교의 횟수 => O(n^2) 
이동의 횟수 => O(n) 

3.삽입 정렬
정렬이 완료된 부분과 완료되지 않은 부분으로 나누고, 정렬 안 된 부분에 있는 데이터를 정렬 된 부분의
특정 위치에 '삽입'해 가는 정렬 방식
비교의 횟수 => O(n^2) 
이동의 횟수 => O(n^2) 

4.힙 정렬
힙의 삽입과 반환을 이용해 정렬하는 방식
O(nlogn/2) 

5.병합 정렬
'분할 정복'이라는 알고리즘 디자인 기법에 근거하여 만들어진 정렬 방법
분할 정복법 3단계
1. 분할 - 해결이 용이한 단계까지 문제를 분할해 나간다.
2. 정복 - 해결이 용이한 수준까지 분할된 문제를 해결한다.
3. 결합 - 분할해서 해결한 결과를 결합하여 마무리한다.
